# **App Name**: NCC Command

## Core Features:

- Role-Based Authentication: Secure login portals for Cadets, Admins, and Managers with password management.
- Cadet Dashboard: Personalized dashboard displaying upcoming events/camps and recent notifications.
- Camp Management: CRUD (Create, Read, Update, Delete) interface for training camps. Admins can view camp details and manage registered cadets.
- Camp Registration Tool: Admins can use a tool to Accept or Reject pending registrations for each camp.
- Attendance Tracking: Mark attendance for cadets for any given date and cadets can view attendance history.
- User Role Management: Managers can add or remove other Admin or Manager users from the system.
- Activity Logging: A system-wide audit log displaying all important actions taken by users.

## Style Guidelines:

- Primary color: Dark blue (#2E4A62) for a professional, military-inspired look.
- Background color: Light gray (#F0F4F7) for a clean, neutral backdrop.
- Accent color: Gold (#D4AF37) for highlights and actionable items, inspired by military insignia.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and readable experience.
- Responsive design with a persistent sidebar for clear navigation.
- Use clean, professional icons to represent various features and actions.
- Subtle transitions and feedback animations for user interactions.
